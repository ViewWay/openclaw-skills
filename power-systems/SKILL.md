## 1. 电力系统基础

### 理论

**三相交流电：**

- **星形 (Y) 连接**: 线电压 $U_L = \sqrt{3} U_{ph}$，线电流 $I_L = I_{ph}$
- **三角形 (Δ) 连接**: 线电压 $U_L = U_{ph}$，线电流 $I_L = \sqrt{3} I_{ph}$
- 三相电压相位差 120°: $u_a = U_m\sin(\omega t)$, $u_b = U_m\sin(\omega t - 120°)$, $u_c = U_m\sin(\omega t - 240°)$

**功率关系：**

| 功率类型 | 公式 | 单位 |
|----------|------|------|
| 有功功率 P | $P = UI\cos\varphi = \sqrt{3} U_L I_L \cos\varphi$ (三相) | W, kW, MW |
| 无功功率 Q | $Q = UI\sin\varphi = \sqrt{3} U_L I_L \sin\varphi$ (三相) | var, kvar, Mvar |
| 视在功率 S | $S = UI = \sqrt{3} U_L I_L = \sqrt{P^2 + Q^2}$ | VA, kVA, MVA |
| 功率因数 PF | $PF = \cos\varphi = P/S$ | — |

**功率三角形:** $S^2 = P^2 + Q^2$，$\varphi = \arctan(Q/P)$

**频率标准:** 50Hz ± 0.5Hz（中国），60Hz ± 0.5Hz（北美/日本部分）

**电压等级（中国）：**

| 等级 | 电压 | 应用 |
|------|------|------|
| 低压 | 220V / 380V | 居民/商业用电 |
| 中压 | 10kV / 35kV | 配电 |
| 高压 | 110kV / 220kV | 输电 |
| 超高压 | 500kV / 750kV | 远距离输电 |
| 特高压 | 1000kV AC / ±800kV DC | 超远距离输电 |

**电力系统拓扑：** 发电 → 升压变电 → 高压输电 → 降压变电 → 配电 → 用电

### 方法

- 三相功率测量: 二表法（3线制）、三表法（4线制）
- 功率因数补偿: 并联电容器组，目标 PF ≥ 0.95
- 电压质量监测: 连续采样 → 统计分析（概率分布、最大/最小/平均值）
- 负荷预测: 时间序列、机器学习

### 工具

- 电能质量分析仪: Fluke 435-II、HIOKI PW3198
- 万用表/钳表: Fluke 87V、HIOKI 3288
- 仿真: MATLAB/Simulink (Simscape Electrical)、PSCAD、ETAP
- 在线监测: SCADA 系统、PMU (相量测量单元)

### 标准

- GB/T 15945-2008: 电力系统频率偏差
- GB/T 12325-2008: 供电电压偏差
- GB/T 15543-2008: 三相电压不平衡
- GB/T 12326-2008: 电压波动与闪变

### 应用

- 配电网络规划与设计
- 工厂供电方案设计（变压器选型、无功补偿）
- 分布式能源并网
- 电力系统仿真与分析

## 2. 电能计量

### 理论

**计量原理（数字式）：**

```
电压/电流采样 → ADC (Σ-Δ) → 数字高通/低通滤波 → DSP 功率计算 → 累积 → 显示/通信
```

**电压/电流有效值 (RMS) 计算：**

$$U_{rms} = \sqrt{\frac{1}{N}\sum_{k=0}^{N-1} u_k^2}$$

$$I_{rms} = \sqrt{\frac{1}{N}\sum_{k=0}^{N-1} i_k^2}$$

**功率计算：**

| 功率 | 公式 |
|------|------|
| 有功功率 P | $P = \frac{1}{T}\int_0^T u(t) \cdot i(t) \, dt \approx \frac{1}{N}\sum_{k=0}^{N-1} u_k \cdot i_k$ |
| 无功功率 Q | $Q = \frac{1}{T}\int_0^T u(t - T/4) \cdot i(t) \, dt$（移相 90° 法） |
| 视在功率 S | $S = U_{rms} \times I_{rms}$ |
| 功率因数 PF | $PF = P / S$（畸变系统: $PF = P / S_1$） |

**四象限功率定义（IEC 62053-23 / IEC 62053-24）：**

```
           Q(+)
            |
      II    |    I
   P(-)Q(+) | P(+)Q(+)
            |
  ---------+----------> P(+)
            |
     III    |    IV
  P(-)Q(-)  | P(+)Q(-)
            |
           Q(-)
```

| 象限 | 有功 | 无功 | 含义 |
|------|------|------|------|
| I | + | + | 正常用电，吸收滞后无功 |
| II | - | + | 发电状态，发出超前无功 |
| III | - | - | 发电状态，发出滞后无功 |
| IV | + | - | 正常用电，吸收超前无功 |

**电能累计：**

$$W = \int_0^T P(t) \, dt \approx \sum_{k} P_k \cdot \Delta t$$

- 翻转处理: 正向有功正向累计，反向有功反向累计或单独记录
- 脉冲输出: 脉冲常数 = imp/kWh，如 1600 imp/kWh → 每个脉冲 1/1600 = 0.000625 kWh

### 方法

- 采样率选择: ≥ 256 点/周期（基波 50Hz → ≥ 12.8kHz）
- Σ-Δ ADC: 过采样 + 数字滤波，典型 OSR = 256-1024
- 窗函数: Hanning 窗减少频谱泄漏
- 滑动平均: 降低功率计算波动
- 死区处理: 小电流/小电压截止（启动电流 typically 0.1% Ib）

### 工具

- 校准台: HY5103、ST9001（三相多功能校验装置）
- 标准表: FLUKE 6100A、K2006（0.01级）
- 电能表检定装置: ZHZ23、CL3112
- 仿真: MATLAB（信号生成 + 计量算法验证）

### 标准

- IEC 62053-21: 静止式有功电能表（0.5S/1/2级）
- IEC 62053-22: 静止式无功电能表
- IEC 62053-23: 静止式无功电能表（2/3级）
- GB/T 17215.321-2021: 电能测量（等同 IEC 62053-21）
- IEC 62052-11: 通用要求

### 应用

- 单相/三相电子式电能表设计
- 多功能电表软件开发
- 计量算法验证与精度优化
- 双向计量（光伏并网/储能）

## 3. 需量测量

### 理论

**需量 (Demand):** 规定时间间隔内的平均功率。

$$D = \frac{1}{T_d} \int_0^{T_d} P(t) \, dt$$

**滑差需量 (Sliding Window Demand):**
- 滑动窗口长度 $T_d$（通常 5min），滑动步长 $\Delta t$（通常 1min）
- 每分钟更新一次需量值
- 计算量: 5min 窗口每分钟滑动 → 5 次重叠计算

**区间需量 (Block Demand):**
- 固定时间间隔: 15min 或 60min
- 按时钟对齐，不重叠

**最大需量 (Maximum Demand):**
- 当前计费周期内所有需量值的最大值
- 通常分有功最大需量和无功最大需量
- 复位: 每月结算日复位

**需量超限告警:**
- 预设需量阈值（如合同容量 80%）
- 超限时记录事件 + 可选继电器动作

### 方法

- 滑差需量实现: 环形缓冲区存储功率采样值 → 每次滑动时更新窗口和
- 区间需量: 整点/半点对齐的固定窗口
- 数据存储: 最大需量 + 发生时间戳 + 最大需量月份累计

### 工具

- 电能表内置需量计算模块
- 上位机采集系统: 需量曲线显示、历史查询

### 标准

- GB/T 17215.301-2021: 多功能电表技术规范
- DL/T 614-2007: 多功能电能表
- 电力用户合同容量约定

### 应用

- 工业用户基本电费计算（按需量或按容量）
- 需量响应与负荷管理
- 变压器容量监控

## 4. 谐波分析

### 理论

**FFT 谐波分析：**

$$X(k) = \sum_{n=0}^{N-1} x(n) e^{-j2\pi kn/N}$$

- 分析范围: 2~50 次谐波（IEC 61000-4-7）
- 采样: 同步采样（锁相倍频）→ 整数周期截断 → 避免频谱泄漏
- FFT 窗长: ≥ 10 个工频周期（50Hz × 10 = 200ms → 采样率 ≥ 2560Hz @ 256点/周期）

**总谐波畸变率 (THD):**

$$THD_U = \frac{\sqrt{\sum_{n=2}^{50} U_n^2}}{U_1} \times 100\%$$

$$THD_I = \frac{\sqrt{\sum_{n=2}^{50} I_n^2}}{I_1} \times 100\%$$

**间谐波 (Interharmonics):**
- 非整数倍基波频率的频率分量
- 检测方法: IEC 61000-4-7 子组/总组方法
- 影响: 电压闪变、滤波器谐振、通信干扰

**GB/T 14549-1993 谐波电流限值（公共电网）：**

| 电压等级 | 基准短路容量 MVA | THDu 限值 | 各次谐波电流限值 |
|----------|------------------|-----------|-----------------|
| 0.38kV | 10 | 5% | 详见标准表 |
| 6/10kV | 100 | 4% | 详见标准表 |
| 35/66kV | 500 | 3% | 详见标准表 |
| 110kV | 750 | 2% | 详见标准表 |

### 方法

- 同步采样: PLL 锁相 50Hz → 倍频生成采样时钟
- FFT 实现: 基-2 FFT，N = 1024 或 2048
- 加窗: Hanning 窗（减少旁瓣泄漏）
- 频率插值: Quinn/McKinney 插值提高频率分辨率

### 工具

- 电能质量分析仪: Fluke 435-II（THD、各次谐波柱状图）
- 电能表内置谐波测量: ADE7880 支持 2~50 次谐波
- 仿真: MATLAB `fft()`、Python `numpy.fft`

### 标准

- GB/T 14549-1993: 电能质量 公用电网谐波
- IEC 61000-4-7: 谐波和间谐波测量方法
- IEEE 519-2022: 电力系统谐波控制

### 应用

- 变频器/整流器负荷谐波评估
- 谐波治理方案设计（有源滤波器 APF）
- 光伏逆变器并网谐波检测
- 电能质量监测与故障诊断

## 5. 电能质量

### 理论

**电压偏差（GB/T 12325-2008）：**

| 电压等级 | 偏差限值 |
|----------|----------|
| 10kV 及以下 | ±7% (额定电压 220V: 207-237V) |
| 35kV 及以上 | 正负偏差绝对值之和 ≤ 10% |

**电压波动与闪变（GB/T 12326-2008）：**

- 电压波动: $d = \frac{\Delta U}{U_N} \times 100\%$
- 短时闪变严重度 $P_{st}$: ≤ 1.0（LV/MV），≤ 0.8（HV）
- 长时闪变严重度 $P_{lt}$: ≤ 0.65（LV/MV），≤ 0.6（HV）
- IEC 闪变测量: 灯-眼-脑模型 → 视感系数加权

**电压暂降/暂升（ITIC/CBEMA 曲线）：**

| 事件 | 残压 | 持续时间 |
|------|------|----------|
| 暂降 | < 90% Un | 0.5 周期 ~ 1min |
| 暂升 | > 110% Un | 0.5 周期 ~ 1min |
| 中断 | < 10% Un | ≤ 3min (瞬时) |

- ITIC 曲线: 设备耐受能力包络线
- 典型原因: 短路故障、大电机启动、雷击

**三相不平衡（GB/T 15543-2008）：**

- 零序不平衡: $\varepsilon_0 = \frac{U_0}{U_1} \times 100\%$
- 负序不平衡: $\varepsilon_2 = \frac{U_2}{U_1} \times 100\%$
- 限值: ≤ 2%（正常运行），≤ 4%（短时）
- 对称分量法: 正序/负序/零序分解

**频率偏差（GB/T 15945-2008）：**
- 正常运行: ±0.2Hz（大系统）/ ±0.5Hz（小系统）
- 事故: ±1.0Hz

### 方法

- 电压质量监测: 每周期采样 → RMS 计算 → 10/12 周期有效值 → 统计分析
- 闪变计算: IEC 61000-4-15 标准算法（模拟灯-眼-脑链）
- 暂降检测: RMS 滑动窗口法、dq 变换法
- 不平衡计算: 对称分量法（Fortescue 变换）

### 工具

- 电能质量监测仪: Fluke 435-II、HIOKI PW3198
- 在线监测: PMU (PMU5000)、电能质量在线监测系统
- 仿真: PSCAD/EMTDC、MATLAB/Simulink

### 标准

- GB/T 12325-2008: 供电电压偏差
- GB/T 12326-2008: 电压波动与闪变
- GB/T 15543-2008: 三相电压不平衡
- GB/T 15945-2008: 电力系统频率偏差
- IEC 61000-4-30: 电能质量测量方法
- IEEE 1159-2019: 电能质量监测推荐实践

### 应用

- 敏感负荷（半导体/医疗/数据中心）供电质量保障
- 电能质量问题诊断与治理
- 配电网电能质量监测网络
- 光伏/储能并网点电能质量评估

## 6. 智能电表功能

### 理论

**分时费率 (TOU, Time of Use)：**

| 费率 | 时间段（典型） | 电价系数 |
|------|---------------|----------|
| 尖峰 | 10:00-12:00, 14:00-17:00 | 1.5× |
| 峰 | 08:00-10:00, 17:00-21:00 | 1.2× |
| 平 | 07:00-08:00, 12:00-14:00, 21:00-23:00 | 1.0× |
| 谷 | 23:00-07:00 | 0.5× |

- 每日费率时段数: 通常 4-8 段
- 支持多套费率方案（工作日/周末/节假日）
- 季费率/月费率切换

**负荷曲线 (Load Profile)：**

| 冻结间隔 | 存储容量（典型） | 存储天数 |
|----------|-----------------|----------|
| 1min | 40 天 | 精细分析 |
| 5min | 200 天 | 常用 |
| 15min | 600 天 | 峰谷统计 |
| 30min | 1200 天 | 长期 |
| 60min | 2400 天 | 概览 |

- 数据块存储: 每块 1 天/1 月
- 冻结方式: 定时冻结、瞬时冻结、日冻结、月冻结、整点冻结

**事件记录：**

| 事件类型 | 触发条件 | 记录内容 |
|----------|----------|----------|
| 断相 | 电压 < 阈值 | 相别、起止时间、期间电量 |
| 过压/欠压 | > 120% / < 80% Un | 电压值、时间 |
| 过流/欠流 | > Imax / < 启动电流 | 电流值、时间 |
| 开盖 | 检测到开盖 | 发生时间 |
| 编程 | 修改参数 | 操作者、内容 |
| 时钟错误 | 电池欠压/被篡改 | 事件时间 |
| 磁场干扰 | 检测到强磁场 | 时间、持续时间 |
| 需量超限 | D > Dlim | 超限值、时间 |
| 功率方向改变 | P 正→负 或 反之 | 时间、功率值 |

**显示：**

- 段码 LCD，典型 8 位数字 + 单位/符号
- 自动轮显: 每隔 5-8 秒切换显示项
- 按键切换: 上翻/下翻/编程
- OBIS 短码: 如 1.8.0（正向有功总电能）、1.8.1（费率1正向有功）

**通信：**

| 接口 | 协议 | 应用 |
|------|------|------|
| RS-485 | DLMS/COSEM、Modbus-RTU | 本地抄表 |
| 红外 | IEC 62056-21 | 现场手持抄表 |
| LoRaWAN | Class A/B/C | 物联网远程抄表 |
| NB-IoT | CoAP/LwM2M | 运营商网络 |
| GPRS/4G | TCP/UDP + DL/T 645/698 | 远程抄表 |
| PLC | DL/T 698.45 | 电力线载波 |

**安全（ESAM 安全模块）：**
- SM2: 非对称加密（数字签名、密钥交换）
- SM3: 哈希算法（数据完整性）
- SM4: 对称加密（数据加密）
- 功能: 数据加密存储、远程安全认证、费控指令签名

**控制：**
- 继电器: 内置磁保持继电器（60A/80A/100A）
- 预付费: 余额不足 → 告警 → 跳闸 → 充值后合闸
- 远程拉合闸: 主站下发命令 → ESAM 验签 → 执行

**时钟：**
- RTC + 温补晶振 (TCXO)，日误差 ≤ 0.5s
- BCD 码存储（秒/分/时/日/周/月/年）
- 对时方式: NTP（网络）、PPS（秒脉冲）、B码（IRIG-B）、广播对时
- 时钟电池: CR2032，寿命 ≥ 10 年

**OTA（双 Bank 固件升级）：**
- Bank A: 当前运行固件
- Bank B: 升级固件下载区
- 校验: CRC32 + 签名验证 → 切换 Bank → 重启

### 方法

- 固件开发: RTOS (FreeRTOS/RT-Thread) + 计量芯片驱动 + 通信协议栈
- 数据存储: Flash 环形缓冲区 + FAT 文件系统
- 低功耗设计: 休眠唤醒、RTC 中断、外设按需开启

### 工具

- 开发环境: Keil MDK / IAR / Eclipse + GCC (ARM Cortex-M)
- 调试: J-Link / ST-Link + 逻辑分析仪
- 抄表软件: 电能表检测软件、DLMS Client（Gurux）
- 测试: 电能表检验装置 + 上位机

### 标准

- GB/T 17215.301-2021: 多功能电表
- DL/T 698.45: 电能信息采集（面向对象协议）
- DL/T 645-2007: 多功能电能表通信协议
- IEC 62056-21: 直接数据交换
- IEC 62056-53: COSEM 应用层
- IEC 62051: 电表标识（ID）

### 应用

- 国网智能电表（单相/三相）
- 分布式光伏并网计量（双向）
- 预付费电表（公寓/宿舍）
- 工业多费率电表
- 出口电表（DLMS/IEC 标准）

## 7. 计量芯片

### 理论

**Σ-Δ ADC 原理：**
- 过采样: 采样率远高于奈奎斯特频率（OSR = 64-1024）
- 量化噪声整形: 将噪声推向高频 → 数字滤波器滤除
- 优点: 高分辨率（16-24bit）、无需精密采样保持

**主流计量芯片对比：**

| 芯片 | 厂商 | 相数 | ADC | 动态范围 | 接口 | 特点 |
|------|------|------|-----|----------|------|------|
| **ATT7022E** | 锐能微 | 三相四线 | 19-bit Σ-Δ | 3000:1 | SPI | 内置有功/无功功率计量 |
| **RN8302B** | 锐能微 | 三相 | 24-bit Σ-Δ | 5000:1 | SPI | 高精度，支持谐波 |
| **RN8615V2** | 锐能微 | 三相 | 24-bit | 10000:1 | SPI | 超高精度 0.1S 级 |
| **ADE7755** | ADI | 单相 | 16-bit Σ-Δ | 1000:1 | 高频脉冲输出 | 经典、低成本 |
| **ADE7880** | ADI | 三相 | 24-bit Σ-Δ | 5000:1 | SPI/I2C | 高精度+谐波分析 |
| **ADE9153A** | ADI | 单相 | 24-bit | 4000:1 | SPI | mSure 自校准 |
| **BL0942** | 贝岭 | 单相 | 16-bit | 1000:1 | 脉冲/UART | 低成本 |
| **HT7033** | 航泰 | 三相 | 24-bit | 5000:1 | SPI | 国产替代 |

**计量芯片典型外围：**
- 电压通道: 电阻分压网络 → 差分输入（660kΩ + 1kΩ）
- 电流通道: 互感器 (CT) 或锰铜微电阻 (Shunt, 100-200μΩ)
- CT 精度: 0.05 级 / 0.1 级，负载电阻 10-50Ω
- 基准电压: 内置或外部 2.5V 基准（温度系数 < 20ppm/°C）

### 方法

- 芯片选型: 精度要求、相数、接口、成本、供应链
- PCB 布局: 计量区模拟地独立 → 单点连接数字地 → 远离开关电源
- 校准: 功率校准（增益/相位/偏移）+ 脉冲常数校准
- 自校准: ADE9153A mSure 技术，无需标准表

### 工具

- 评估板: 各厂商提供 EVB
- 校准软件: 厂商校准工具（ATT7022E → ATT7022E_Calibration_Tool）
- 采集卡: 高精度 ADC 采样 → PC 分析

### 标准

- GB/T 17215.211-2021: 计量芯片通用要求
- 各芯片 datasheet 中的精度指标

### 应用

- 三相多功能电表: RN8302B/RN8615V2/ADE7880

## 8. 校准

### 理论

**比较法（标准表法）：**

被检表读数与标准表读数对比，计算相对误差:

$$\gamma = \frac{E_x - E_0}{E_0} \times 100\%$$

- $E_x$: 被检表电能读数
- $E_0$: 标准表电能读数

**功率校准参数：**

| 参数 | 校准项 | 方法 |
|------|--------|------|
| 电压增益 | 电压通道比例系数 | 标准电压源 → 读取 ADC 值 → 计算系数 |
| 电流增益 | 电流通道比例系数 | 标准电流源 → 读取 ADC 值 → 计算系数 |
| 相位补偿 | 电压/电流通道相位差 | 功率因数 0.5L/0.5C → 调整相位校准字 |
| 偏移补偿 | 零功率/零电流偏置 | 断流/短路 → 读取偏移值 → 补偿 |

**脉冲常数校准：**
- 理论脉冲数 = 测量电能 × 脉冲常数
- 实测脉冲与理论值对比 → 调整脉冲常数寄存器

**启动试验:**
- 条件: 电压 Un，功率因数 1.0
- 负载电流: 0.1% Ib（0.2S 级）/ 0.2% Ib（0.5S 级）/ 0.4% Ib（1 级）
- 要求: 电表能连续计量并输出脉冲

**潜动试验:**
- 条件: 电压 1.15Un，电流回路断开
- 要求: 测试时间内输出脉冲 ≤ 1 个

**精度等级：**

| 等级 | 有功误差限 | 无功误差限 | 应用 |
|------|-----------|-----------|------|
| 0.1S | ±0.1% | — | 关口表/标准表 |
| 0.2S | ±0.2% | — | 关口表 |
| 0.5S | ±0.5% | ±2.0% | 大工业 |
| 1.0 | ±1.0% | ±2.0% | 一般工商业 |
| 2.0 | ±2.0% | ±3.0% | 居民 |

### 方法

- 多点校准: Un × {0.01Ib, 0.05Ib, 0.1Ib, 0.5Ib, Ib, Imax} × {1.0, 0.5L, 0.8C}
- 温度系数测试: -10°C / 20°C / 45°C 全量程
- 频率影响测试: 49Hz / 50Hz / 51Hz
- 谐波影响测试: 各次谐波叠加

### 工具

- 三相校验装置: HY5103（0.05级）、ST9001（0.01级）
- 单相校验台: YC-1893D（0.1级）
- 标准功率表: K2006（0.01级）
- 温度试验箱: 高低温交变试验箱

### 标准

- JJG 596-2012: 电子式交流电能表检定规程
- GB/T 17215.321-2021: 电能测量准确度
- DL/T 825-2002: 电能计量装置现场检验规程

### 应用

- 生产线电表校准（自动化校准台）

## 10. 电力市场

### 理论

**分时电价（峰谷电价）：**

| 时段 | 电价（典型） | 目的 |
|------|-------------|------|
| 尖峰 | 1.2-1.5 元/kWh | 削峰 |
| 高峰 | 0.8-1.0 元/kWh | 移峰 |
| 平段 | 0.5-0.7 元/kWh | 基准 |
| 低谷 | 0.2-0.4 元/kWh | 填谷 |

**需量电价:**
- 按最大需量计费: 基本电费 = 最大需量 × 单价（如 38 元/kW·月）
- 按变压器容量计费: 基本电费 = 变压器容量 × 单价（如 25 元/kVA·月）
- 用户可选: 选择较低费用的方式

**分布式光伏上网计量：**

```
光伏 → 逆变器 → 并网点
├── 自发自用: 光伏发电 → 本地负载消耗
├── 余电上网: 多余电能 → 电网（上网电量计量）
└── 全额上网: 全部发电 → 电网
```

- 需要双向计量电表
- 上网电价: 脱硫煤标杆电价 + 补贴（逐渐退坡）

**储能双向计量：**

```
电网 ←→ 双向电表 ←→ 储能变流器(PCS) ←→ 电池
         ↓
    充电计量（正向有功）
    放电计量（反向有功）
```

- 四象限电表: 分别记录充放电电量
- 应用: 峰谷套利、需求响应、调频调峰

**碳排放计量：**
- 碳排放因子: 每kWh电量对应的CO₂排放量（中国电网平均 ~0.58 tCO₂/MWh）
- 绿电交易: 可再生能源证书（REC）
- 碳市场: 碳排放权交易

### 方法

- 电费计算: 电度电费 + 基本电费 + 力调电费（无功考核）
- 力调电费: PF < 0.9 加收电费，PF > 0.95 减收电费
- 需求响应: 提前通知 → 用户降低负荷 → 补偿

### 工具

- 营销系统: 电力公司计费系统
- 能源管理系统 (EMS): 园区/建筑能源监控
- 交易平台: 电力现货市场、碳交易市场

### 标准

- 发改价格〔2021〕1439号: 分时电价政策
- GB/T 32127-2015: 电力需求响应系统
- 国家碳市场交易规则

### 应用

- 工业用户电费优化（避峰就谷）
- 分布式光伏+储能经济性分析
- 虚拟电厂 (VPP) 聚合
- 碳资产管理
