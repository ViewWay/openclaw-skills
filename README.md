## 🚀 使用方法

### OpenClaw（原生支持）

```bash
# 克隆到 workspace/skills 目录
cd ~/.openclaw/workspace
git clone https://github.com/ViewWay/openclaw-skills.git /tmp/openclaw-skills
cp -r /tmp/openclaw-skills/* skills/
openclaw skills check
```

Skills 会自动被 OpenClaw 识别和加载。

#### 方法二：项目级配置


### Cursor

```bash
# 1. 复制到项目
cp -r ~/claw-skills /path/to/project/.claw-skills

# 2. 创建 .cursorrules
cat > /path/to/project/.cursorrules << 'EOF'
## Knowledge Base
When answering domain-specific questions, read the relevant SKILL.md files
under .claw-skills/ directory first.

### Key references:
- Communication: .claw-skills/communication-protocols/SKILL.md
- Embedded: .claw-skills/embedded-arm/SKILL.md
- Algorithms: .claw-skills/data-structures-algorithms/SKILL.md
- CS Index: .claw-skills/computer-science-knowledge-base/SKILL.md
- Math: .claw-skills/math/SKILL.md
- Physics: .claw-skills/physics/SKILL.md

Read the file before answering to ensure accuracy.
EOF
```

### 其他 AI 编程工具（通用方法）
