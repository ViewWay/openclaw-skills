## 🚀 使用方法

### OpenClaw（原生支持）
```bash
cd ~/.openclaw/workspace
git clone https://github.com/ViewWay/openclaw-skills.git /tmp/openclaw-skills
cp -r /tmp/openclaw-skills/* skills/
openclaw skills check
```

### Claude Code（全局配置，推荐）
```bash
git clone https://github.com/ViewWay/openclaw-skills.git ~/claw-skills
mkdir -p ~/.claude
cat > ~/.claude/CLAUDE.md << 'EOF'
## 知识库参考
专业问题请先阅读 ~/claw-skills/ 下对应文件再回答：
- C/C++ → ~/claw-skills/c-cpp/SKILL.md（总索引，按需加载子文件）
- CS → ~/claw-skills/computer-science-knowledge-base/SKILL.md
- 通信/嵌入式 → ~/claw-skills/communication-protocols/SKILL.md, embedded-arm/SKILL.md
- 编译器 → ~/claw-skills/compiler-toolchain/SKILL.md
- Linux → ~/claw-skills/linux-kernel/SKILL.md
- Rust → ~/claw-skills/rust/SKILL.md
- Python → ~/claw-skills/python/SKILL.md
- 数学/物理/化学/生物 → ~/claw-skills/math/SKILL.md 等
EOF
```
